# E-Commerce SQL Data Analysis

## About the Project
Analyzed e-commerce sales data using MySQL to find
revenue trends, top products, and customer behavior.

## Tools Used
- MySQL
- MySQL Workbench
- Kaggle Dataset

## Key Findings
- Clothing category generates highest revenue
- COD is most popular payment method
- November has peak sales (festive season)

## Queries Covered
- Aggregations (SUM, AVG, COUNT)
- GROUP BY and HAVING
- ORDER BY and LIMIT
- Date functions (MONTH, QUARTER)
- Filtering with WHERE
